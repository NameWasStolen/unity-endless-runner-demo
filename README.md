# unity-endless-runner-demo

A simple endless runner prototype built in Unity as part of a workshop to teach fundamental Unity and game development concepts.

## Overview
This project was developed as a small demo for a Unity workshop, focusing on the structure of a basic game loop and interactive mechanics in a clear, minimal setup.

## Features
- Player movement and controls  
- Obstacle spawning and collision handling  
- Score tracking  
- Basic game state management (start / game over)  

## Tech Stack
- Unity (Game Engine)  
- C# (Scripting)  

## Purpose
Designed as an introductory example to help beginners understand basic Unity concepts and features. 

## Notes
This is a lightweight prototype focused on functionality and learning outcomes rather than polished visuals or full game design.
